# RedBot Media Player Cog Repo

This repository is the standalone source of truth for the `ha_red_rpc` Red cog used by RedBot Media Player.

## Install (Red Downloader)

```text
[p]repo add redbot_media_player_cog https://github.com/AtticusG3/redbot-media-player-cog
[p]cog install redbot_media_player_cog ha_red_rpc
[p]load ha_red_rpc
```

## Local path install

```text
[p]addpath /share/redbot_cogs
[p]load ha_red_rpc
```

If you use the Home Assistant add-on in `redBot-hass`, `/share/redbot_cogs/ha_red_rpc` is seeded automatically and can auto-sync from this repo.
